Use of print_sorted

add module
	>>> MyList = __import__('1-my_list').MyList

create variable and add values to list
	>>> my_list = MyList()
	>>> my_list.append(1)
	>>> my_list.append(4)
	>>> my_list.append(2)
	>>> my_list.append(3)
	>>> my_list.append(5)

print original list
	>>> print(my_list)
	[1, 4, 2, 3, 5]

print sorted list
	>>> my_list.print_sorted()
	[1, 2, 3, 4, 5]

add negative values and sort
	>>> my_list.append(-1)
	>>> my_list.append(-9)
	>>> my_list.print_sorted()
	[-9, -1, 1, 2, 3, 4, 5]

print_sorted returns a new list
	>>> new_list = my_list.print_sorted()
	[-9, -1, 1, 2, 3, 4, 5]

empty list and sort
	>>> my_list.clear()
	>>> my_list.print_sorted()
	[]
